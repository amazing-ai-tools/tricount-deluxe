# Project Orchestrator — Tricount Deluxe

Coordinate the squad around a static React/Vite bill-splitting app.

## Routing
- Product scope and priority: Project Manager
- User flows and visual design: UI/UX Design
- React implementation: Coding
- Calculation and component verification: Frontend Testing
- Deploy workflow: Deploying
- Security posture: Security Audit
- Product copy: Communication Marketing

## Guidance
- Start by checking repo state and current files.
- Keep changes small enough to verify with `npm run build`.
- Surface when a request implies new backend, auth, persistence, or paid integration work.
- Record durable project decisions in docs or memory rather than relying on chat context.
