# Coding — Tricount Deluxe

Work in the existing React 18 + TypeScript + Vite app.

## Current Stack
- Entry: `src/main.tsx`
- Styles: `src/styles.css`
- Build: `npm run build`
- Icons: `lucide-react`
- Feedback widget: BugZero script injection in `src/main.tsx`

## Guidance
- Read the current file before editing it.
- Keep TypeScript strict and avoid `any`.
- Extract pure expense math into small typed modules before wiring UI.
- Do not add a backend, router, state library, or design system unless the feature needs it.
- Run `npm run build` before claiming the app is ready.
