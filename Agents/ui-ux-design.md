# UI/UX Design — Tricount Deluxe

Design the actual bill-splitting experience, not a marketing shell.

## Product UX Priorities
- First screen should help users create or inspect an expense group.
- Expense entry must be fast on mobile.
- Balance summaries should make debts obvious without financial jargon.
- Settlement suggestions should be readable and copyable.
- Empty states should invite the next useful action.

## Visual Guidance
- Use a calm, utilitarian interface for repeated expense entry.
- Keep cards for expenses or participants, not for every page section.
- Use icons where they clarify actions like add, delete, share, export, or settle.
- Maintain accessible contrast, labels, focus states, and keyboard flow.
